# Skill: Activity Recommendations

## Purpose
Provide personalized activity recommendations based on family profile, preferences, and past visits.

## When to Use
- User asks: "What should we do this weekend?"
- User needs a suggestion: "Find something fun for today"
- User wants recommendations matching preferences
- Suggesting activities for a new family member's age
- Building a weekend itinerary

## How It Works

Recommendations are generated by analyzing:

1. **Family Profile**
   - Family member ages (focus on toddler age)
   - Home city and willing travel distance
   - Preferred activity categories
   - Budget preferences
   - Accessibility needs (stroller-friendly, wheelchair access, etc.)
   - Indoor/outdoor preference

2. **Visit History**
   - Past activities visited (to avoid repeats)
   - Ratings given (avoid low-rated categories)
   - Time since last visit (recommend variety)
   - Notes about experiences

3. **Current Activity Database**
   - Activities available in home city or travel range
   - Enriched details (hours, pricing, facilities)
   - Age suitability for current child age
   - Status (new, enriched, visited, favorite)

## Recommendation Scoring

Activities are scored on:

```
Score = (Base Match) × (Preference Weight) × (Variety Factor) × (Availability)

Where:
- Base Match: How well activity matches family profile (0-100)
- Preference Weight: Alignment with family preferences (0-1.0)
- Variety Factor: Penalty for recent visits, boost for rarely-visited categories (0-1.0)
- Availability: Is it open now? Does it fit budget? (0-1.0)
```

## Scoring Factors Detail

### Base Match (0-100)
- Age suitability: +20 if perfect age range, 0 if outside range
- Category preference: +15 if in preferred categories
- Budget alignment: +15 if matches budget preference
- Accessibility: +10 if has required facilities
- Travel distance: +20 if within max_travel_minutes, -10 if too far
- Indoor/outdoor: +10 if matches preference
- Toddler-friendly: +10 if activity is toddler-friendly

### Preference Weight (0-1.0)
- Favorite category: 1.0
- Preferred category: 0.8
- Neutral category: 0.5
- Non-preferred category: 0.2
- Excluded category: 0.0

### Variety Factor (0-1.0)
```
If activity visited in last 30 days: 0.3
If activity visited 30-60 days ago: 0.6
If activity visited >60 days ago: 0.9
If activity never visited: 1.0

If category visited recently (multiple times this month): 0.4
If category hasn't been visited this month: 1.0
```

### Availability (0-1.0)
- Open right now: 1.0
- Opens within 1 hour: 0.9
- Opens within 4 hours: 0.7
- Closed today: 0.1
- Budget affordable: 1.0
- Over budget: 0.5
- Explicitly excluded: 0.0

## Best Practices for Recommendations

1. **Always check recency** — Avoid suggesting activities visited in last 7 days
2. **Consider child development** — A 2-year-old needs different activities than a 5-year-old
3. **Respect family preferences** — Weight toward preferred categories
4. **Suggest variety** — Don't always recommend the same playground
5. **Be practical** — Consider travel time, cost, current time/weather
6. **Personalize with history** — Activities with high ratings in similar categories score higher
7. **Provide reasoning** — Explain *why* each activity is recommended

## Personalization Rules

### For Age 0-1 (Babies/Infants)
- Prioritize: outdoor parks, nature activities, libraries
- Avoid: loud/crowded venues, intense activities
- Require: stroller-friendly, changing facilities, quiet areas

### For Age 2-3 (Toddlers)
- Prioritize: playgrounds, parks, indoor play spaces, swimming
- Require: stroller-friendly, changing facilities, café/rest areas
- Prefer: short-medium activities (30-120 minutes)
- Avoid: museums with fragile objects, overly crowded venues

### For Age 4-5 (Preschool)
- Expand: museums, animal attractions, shows, sports
- Preference shift: More structured activities
- Duration: Can handle 2-3 hour activities

## Handling Preferences

### Budget-Conscious Families
- Score free activities highest
- Group activities by cost tier
- Highlight family deals

### Accessibility-First
- Only recommend stroller-friendly for families with toddlers
- Filter by required facilities
- Calculate travel times realistically

### Outdoor-Preferring
- Boost outdoor activities
- Provide indoor alternatives for bad weather
- Note seasonal variations

### Variety-Seeking
- Strong variety penalty for recent visits
- Mix categories in recommendations
- Suggest unexplored activity types
