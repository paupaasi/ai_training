# Chip Strategy

## Purpose
Advise on optimal timing for using FPL chips (Wildcard, Free Hit, Bench Boost, Triple Captain).

## When to Use
- User asks "when should I use my wildcard?"
- User wants chip strategy advice
- User has all chips available

## Tools Required
- fpl_fetcher (get gameweeks/events)
- fixture_difficulty (identify double gameweeks)

## FPL Chips Available
1. **Wildcard (WC)** - Unlimited transfers for one gameweek, permanent
2. **Free Hit (FH)** - Unlimited transfers for one gameweek, reverts after
3. **Bench Boost (BB)** - All 15 players score points
4. **Triple Captain (TC)** - Captain scores 3x instead of 2x

## Optimal Strategy
- **Wildcard** → Before big double gameweek (GW32-35)
- **Bench Boost** → In the double gameweek (GW33 or GW36)
- **Free Hit** → Blank gameweek
- **Triple Captain** → Double gameweek with premium player

## Chip Combination Strategies
1. WC32 → BB33 → FH34 (popular)
2. FH33 → WC35 → BB36 (if missing DGW33)
3. WC34/35 → BB36 (late season)

## Key Considerations
- Current rank and chip availability
- Team value (WC costs ITB if valuable)
- Upcoming fixture swings
- Double gameweek schedule

## Example
```
User: "When should I use my chips?"
-> Get current gameweek -> Identify DGWs -> Recommend optimal timing
```
