    Työehtosopimukset

    Rakennusliitto neuvottelee työnantajaliittojen kanssa yleissitovat työehtosopimukset 7 alalle. Sopimuksissa sovitaan minimiehdoista, joilla rakennusaloilla tehdään työtä.

    Työehtosopimuksessa eli TES:ssä määritellään muun muassa alan palkat, työajat, lomakorvaukset, lomien pituudet, työajan lyhennykset, arkipyhäkorvaukset ja sairausajan palkat.

    Kaikki seitsemän työehtosopimusta ovat yleissitovia. Työehtosopimusten yleissitovuudella tarkoitetaan sitä, että kaikki Suomessa toimivat yritykset ovat velvollisia noudattamaan alan minimityöehtoja.

    Laissa säädetyt minimiehdot koskevat kaikkia työntekijöitä alasta riippumatta. Työehtosopimuksissa voidaan sopia paremmista ehdoista kuin mitä laissa lukee.
